# Documents

This directory contains miscellaneous documentation that is not strictly an "article" (e.g., project documentation, internal notes, memos).

## How to Create a Document

Use the helper script:

```powershell
./scripts/new-document.ps1 -Title "My Document"
```

## How to Build

```powershell
./scripts/build.ps1 -Name "my-document" -Collection "documents"
```
