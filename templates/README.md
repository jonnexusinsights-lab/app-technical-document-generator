# Templates

This directory contains LaTeX templates used by Pandoc to generate PDFs.

## Files

- `editorial.tex`: The main template for editorial-quality PDF generation. It includes configuration for fonts, layout, syntax highlighting, and more.

## Customization

You can modify `editorial.tex` to change:
- Page margins
- Fonts (requires `xelatex` or `lualatex` if using system fonts)
- Header/Footer styles
- Package inclusions
